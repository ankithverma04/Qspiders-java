# Day 1

**Work Summary (≈100 words)**
On the first day of the internship, I was introduced to the fundamentals of Java operators. The session focused on understanding different types of operators used in Java programming such as arithmetic, relational, logical, and assignment operators. These operators help perform mathematical calculations, comparisons, and logical operations within programs. I learned how arithmetic operators are used for performing addition, subtraction, multiplication, division, and modulus operations. Relational operators were used to compare values and produce boolean results. Logical operators were used to combine multiple conditions. I also practiced simple Java programs to understand how these operators work in real programming scenarios.

**Learning Outcomes (≈100 words)**
From this session, I gained a clear understanding of the different categories of operators in Java and their role in programming. I learned how arithmetic operators help perform calculations and how relational operators are used to compare values in decision making. Logical operators helped me understand how multiple conditions can be evaluated together. I also learned about assignment operators and how they simplify variable value updates in programs. Practicing simple programs improved my understanding of operator precedence and expression evaluation. Overall, the session helped me build a strong foundation in using operators effectively while writing Java programs.

I practiced writing programs using if, if-else, and if-else ladder statements to handle multiple scenarios. # Day 3

**Work Summary (≈100 words)**
During the third day of the internship, the focus was on the switch-case statement and the for loop in Java. The switch-case statement was introduced as an alternative to multiple if-else conditions when dealing with multiple choices or menu-driven applications. I learned how switch expressions are evaluated and how different cases are executed depending on the value. The for loop was also discussed as a looping construct used to repeat a block of code a specific number of times. Several example programs were implemented to understand how loops work and how repetitive tasks can be handled efficiently in Java programs.

**Learning Outcomes (≈100 words)**
Through this session, I learned how the switch-case statement simplifies multi-way decision-making in Java programs. I understood how different cases are evaluated and how the break statement helps control program execution. I also learned the structure of the for loop, including initialization, condition, and increment sections. Practicing programs helped me understand how loops are used to repeat operations efficiently. This session strengthened my understanding of program flow and improved my ability to implement iterative logic in Java programming.

# Day 4

**Work Summary (≈100 words)**
On the fourth day of the internship, I studied different looping statements in Java including while loop, do-while loop, and nested loops. The session explained how loops are used to execute a block of code repeatedly until a condition becomes false. I learned that while loops check the condition before executing the loop body, whereas do-while loops execute the code at least once before checking the condition. Nested loops were also introduced, where one loop is placed inside another loop. Practical programs such as pattern printing, multiplication tables, and repeated calculations were implemented to understand how looping statements work.

**Learning Outcomes (≈100 words)**
From this session, I gained a deeper understanding of looping constructs and how they help automate repetitive tasks in programming. I learned when to use while loops and do-while loops depending on the program requirements. Understanding nested loops helped me see how complex patterns and matrix-style operations can be implemented. Practicing different loop-based programs improved my logical thinking and programming skills. This session helped me understand the importance of loops in handling repetitive operations efficiently in real-world software applications.
