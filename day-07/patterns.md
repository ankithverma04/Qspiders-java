# Day 7 – Pattern Programs (Easy Level)

## Objective
To understand how pattern programs are implemented in Java using nested loops and to practice printing simple patterns using symbols and numbers.

---

# Introduction

Pattern programs are commonly used to improve logical thinking and strengthen the understanding of loops in programming. These programs involve printing characters, numbers, or symbols in a structured format to create visual patterns.

Pattern printing mainly depends on the use of nested loops. By adjusting the number of iterations in the loops, different shapes and structures can be created.

---

# Concept of Pattern Printing

Pattern programs mainly use:

- Nested loops
- Print statements
- Loop control conditions

In most pattern problems:
- The **outer loop** controls the number of rows.
- The **inner loop** controls the number of elements printed in each row.

By modifying the loop conditions, programmers can generate a variety of patterns such as triangles, squares, pyramids, and number patterns.

---

# Easy Level Pattern Programs

Easy pattern programs generally involve simple shapes or sequences using stars or numbers.

Examples include:

- Square star pattern
- Right triangle star pattern
- Inverted triangle pattern
- Number triangle pattern
- Repeated number pattern

These patterns help beginners understand how nested loops operate and how program output can be controlled.

---

# Importance of Pattern Programs

Pattern programs help programmers:

- Improve logical thinking
- Understand nested loop execution
- Practice loop control
- Develop problem-solving skills

They are commonly used in programming practice and technical interview preparation.
